# تطبيق البحث في اليوتيوب (YouTube Search App)

هذا هو تطبيق الأندرويد الخاص بي لتسليم الواجب الثاني (Assignment 2). يقوم التطبيق بالاتصال بواجهة برمجة تطبيقات اليوتيوب الرسمية (YouTube v3 API) للبحث عن الفيديوهات وعرضها في قائمة مرتبة ونظيفة للمستخدم.

## تفاصيل ومميزات المشروع:
- **خاصية البحث:** يمكن للمستخدم كتابة أي كلمة بحث في شريط البحث لجلب الفيديوهات المتعلقة بها مباشرة من سيرفرات اليوتيوب.
- **الاتصال والشبكات (Retrofit & Threading):** استخدمت مكتبة **Retrofit** للتعامل مع طلبات الشبكة. يتم إرسال طلب الـ API بشكل غير متزامن في الخلفية باستخدام دالة (`call.enqueue`) حتى تظل واجهة التطبيق سلسة ولا تتجمد أثناء التحميل.
- **عرض القائمة والصور (Glide & RecyclerView):** يتم عرض نتائج البحث (10 فيديوهات) باستخدام الـ `RecyclerView`. وقمت بدمج مكتبة **Glide** لكي تقوم بتحميل وعرض وتخزين الصور المصغرة للفيديوهات (Thumbnails) بكفاءة وسرعة.
- **الميزة الإضافية لزيادة الدرجات (Extra Credit):** قمت بإضافة ميزة ترتيب الفيديوهات عبر تمرير متغير `"viewCount"` لطلب الـ API، مما يجعل النتائج تترتب تلقائياً من الفيديو الأعلى مشاهدة إلى الأقل مشاهدة.
- **معالجة الأخطاء (Error Handling):** يقوم التطبيق بفحص صندوق البحث؛ إذا كان فارغاً تظهر رسالة تنبيهية للمستخدم (`Please enter a search word`). كما يعالج التطبيق مشكلة انقطاع الإنترنت بنجاح داخل دالة `onFailure`.

## طريقة التشغيل والتجهيز:
1. قم بتحميل أو عمل Clone لهذا المستودع.
2. افتح المشروع باستخدام برنامج **Android Studio**.
3. تأكد من وجود مفتاح فعال في متغير الـ `API_KEY` داخل ملف `MainActivity.java`.
4. اعمل مزامنة للملفات (Sync Gradle) ثم اضغط على زر التشغيل (Run) لتثبيت التطبيق.
